# 🌌 NovaSphere

A modern knowledge discovery mobile app built with React Native, Expo, and Express.js that allows users to explore millions of topics using Wikipedia APIs with a beautiful and premium UI experience.

---

## ✨ Features

- 🔍 Smart topic search
- 📚 Detailed article pages
- 🖼️ Rich images & galleries
- 🕘 Recent searches
- 📱 Modern mobile UI
- ⚡ Fast and lightweight
- 🌙 Dark theme design
- 📖 Wikipedia-powered content
- 🎨 Custom splash screen
- 🔗 GitHub APK Releases
